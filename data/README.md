# Data

Bundled demo data:

```text
data/demo/scaffold_from_scratch/       processed molecule tables; scaffold folds are regenerated
data/demo/datasail_esa_cases/          raw DataSAIL splits for Ames, ESOL, and pKb
```

Full processed datasets, splits, and prediction tables are archived at <https://doi.org/10.5281/zenodo.20337102>. Place or symlink them as:

```text
data/processed/
data/splits/
data/predictions/
```
